Git相关命令

git init ：初始化本地git目录，生成 .git 文件夹

git add:git add命令将文件内容添加到索引(将修改添加到暂存区)。也就是将要提交的文件的信息添加到索引库中。

git commit -m 消息:提交，提交信息为-m后面这段信息

git commit -v:提交，并启动vim，显示本次有哪些修改。
